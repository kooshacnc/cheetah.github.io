Cheetah is an open source and free Download Manager. Supports GNU/Linux Distributions, BSDs, macOS and Microsoft Windows operating systems. It has recovery and resume capabilities to restore the interrupted downloads. You can join Cheetah contributers & Help us with developing it.

Notable Features:

  • Http, Https Support for now

  • Divides downloads into multiple streams for faster downloading

  • Create many connection for download default 8 connection

  • Redownload ability & Rejoin ability

  • Themes
